-- Gamepad Controls

Movement - gamepad 1
    - all sticks
    - right_trigger (for slow movement)

Shooter control - gamepad 1
    - left_bumper (shooting motors)
    - a (shoot once)
    - b (shoot three times)

Intake - right_bumper gamepad1
Outtake - right_bumper gamepad2

WobbleArmControl - gamepad2
    - dpad_up
    - dpad_right
    - dpad_down

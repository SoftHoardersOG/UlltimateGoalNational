    - right_trigger (for slow movement)

Shooter control - gamepad 1 
    - left_bumper (shooting motors) ~~~~~ TeleOperated/Shooter

~~~~~ TeleOperated/ChangeShootingAngle
    - a (shoot once)
    - b (shoot three times)

~~~~~~ TeleOperated/Intake

Outtake - right_bumper gamepad2

~~~~~~ TeleOperated/Wobble
